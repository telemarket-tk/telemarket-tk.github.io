## Installation

**Jekyll install**

```sh
$ gem install jekyll
Fetching: liquid-3.0.6.gem (100%)

Fetching: jekyll-3.1.2.gem (100%)

```

**New site**

```sh
$ jekyll new . --force
```

`.`: Create from within the directory  

`--force`: Override *folder exists and is not empty* conflict message

**Serve site**

```sh
$ jekyll s
```
